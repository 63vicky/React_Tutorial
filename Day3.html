ЁЯзСтАНЁЯПл React Class Plan тАУ Day 3

Topics: Events, Conditionals, Lists

Duration: 1 Hour

Goal: Students should be able to make UI interactive and learn to display data with loops + logic.

тП▒я╕П 0 тАУ 10 mins: Recap + Intro

First 2 minutes: Ask students where they used useState and useEffect previously.

Then explain that today we'll make React interactive and dynamic - like changing something on button click, or displaying lists.

тП▒я╕П 10 тАУ 25 mins: React Events

ЁЯза Explain:

How onClick, onChange, onSubmit work (camelCase in JSX)

ЁЯФз Demo Task:

function handleClick() {
  alert("Button clicked!");
}

return <button onClick={handleClick}>Click Me</button>;

ЁЯзк Real task:

Create a button that increases count on click and logs "Clicked" to console.

тП▒я╕П 25 тАУ 40 mins: Conditional Rendering

ЁЯза Explain:

Showing conditions in JSX using if, ? :, && operators

ЁЯФз Demo Task:

const isLoggedIn = true;

return (
  <div>
    {isLoggedIn ? <h2>Welcome Back!</h2> : <h2>Please Login</h2>}
  </div>
);

ЁЯзк Task:

Create a state: show, and toggle true/false with button click.

When show is true, display "Hello User", otherwise show nothing.

тП▒я╕П 40 тАУ 55 mins: List Rendering

ЁЯза Explain:

Converting arrays to components using .map()

Why key prop is important?

ЁЯФз Demo Task:

const fruits = ["Apple", "Banana", "Mango"];

return (
  <ul>
    {fruits.map((item, index) => (
      <li key={index}>{item}</li>
    ))}
  </ul>
);

ЁЯзк Task:

Create an array with student names and display them in a list.

тП▒я╕П 55 тАУ 60 mins: Q&A + Homework

Answer any student doubts

Homework:

  • Create a task list that shows complete/incomplete conditionally
  • Bonus: Add remove button to list items

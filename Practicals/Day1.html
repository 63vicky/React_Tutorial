üß© Task 3: Create a Class Component

Create a new file: WelcomeClass.jsx

Write the following class component in it:

import React, { Component } from "react";

class WelcomeClass extends Component {
  render() {
    return <h3>Welcome from Class Component!</h3>;
  }
}

export default WelcomeClass;

Import and use it in App.jsx:

import WelcomeClass from "./WelcomeClass";

function App() {
  return (
    <div>
      <WelcomeClass />
    </div>
  );
}

üìå Expected Output:

Welcome from Class Component!

React Router Practicals for Students

Note: All solutions are provided in separate files. Click the "View Solution" links to see each implementation.

Practical 1: Basic Routing Setup

Task:

  • Create three components: Home, About, and Contact.
  • Set up routing using react-router-dom.
  • Show navigation links to switch between these pages without refreshing the page.

Expected Output:

  • When you click on "About", the About component content should render.
  • When you click on "Contact", the Contact component should render.

Setup Instructions:

  1. Create a React project using create-react-app
  2. Install react-router-dom: npm install react-router-dom
  3. Create a solutions folder in your project directory
  4. Save each solution file as practical1.html, practical2.html, etc. in the solutions folder

Practical 1: Controlled Text Input

Task: Create a text input where users can enter their name.

एक text input बनाइए जिसमें user नाम लिख सके।

function NameInput() {
        const [name, setName] = React.useState('');

        return (
            <div>
            <input
                type="text"
                value={name}
                onChange={(e) => setName(e.target.value)}
                placeholder="Enter your name"
            />
            <p>Your name is: {name}</p>
            </div>
        );
        }

Key Points:

  • Uses useState to manage the input value
  • The input is controlled - its value comes from React state
  • onChange updates the state with each keystroke

Practical 2: Form Submission

Task: Create a form with name and email inputs that shows an alert on submit.

नाम और ईमेल input वाला form बनाइए जो submit पर alert दिखाए। 

function SignupForm() {
  const [formData, setFormData] = React.useState({
    name: '',
    email: ''
  });

  const handleSubmit = (e) => {
    e.preventDefault();
    alert(`Submitted: ${formData.name}, ${formData.email}`);
  };

  return (
    <form onSubmit={handleSubmit}>
      <input
        type="text"
        placeholder="Name"
        value={formData.name}
        onChange={(e) => 
          setFormData({...formData, name: e.target.value})
        }
      />
      <input
        type="email"
        placeholder="Email"
        value={formData.email}
        onChange={(e) => 
          setFormData({...formData, email: e.target.value})
        }
      />
      <button type="submit">Submit</button>
    </form>
  );
}

Key Points:

  • Form data stored in a single state object
  • preventDefault() stops page reload on submit
  • Each input has its own onChange handler

Practical 3: Multiple Inputs with Single Handler

Task: Create username/password fields with one onChange handler.

Username और Password fields को एक ही onChange handler से manage करें। 

function LoginForm() {
  const [formData, setFormData] = React.useState({
    username: '',
    password: ''
  });

  const handleChange = (e) => {
    const {name, value} = e.target;
    setFormData(prev => ({...prev, [name]: value}));
  };

  return (
    <form>
      <input
        name="username"
        type="text"
        value={formData.username}
        onChange={handleChange}
      />
      <input
        name="password"
        type="password"
        value={formData.password}
        onChange={handleChange}
      />
    </form>
  );
}

Key Points:

  • Uses name attributes to identify inputs
  • Single handler updates the correct field using computed property names
  • Functional update ensures latest state

Practical 4: Dropdown Selector

Task: Create a language selector dropdown.

Language selector dropdown बनाइए जो selected language दिखाए। 

function LanguageSelector() {
  const [language, setLanguage] = React.useState('JavaScript');

  return (
    <div>
      <select 
        value={language}
        onChange={(e) => setLanguage(e.target.value)}
      >
        <option value="JavaScript">JavaScript</option>
        <option value="Python">Python</option>
        <option value="Java">Java</option>
      </select>
      <p>Selected: {language}</p>
    </div>
  );
}

Key Points:

  • Dropdown is controlled via value prop
  • Initial value set in useState
  • Works just like text inputs but with <select>

Practical 5: Checkbox Input

Task: Create a checkbox that shows status when toggled.

Checkbox बनाइए जो status दिखाए जब toggle किया जाए।

function AgreementCheckbox() {
  const [agreed, setAgreed] = React.useState(false);

  return (
    <div>
      <label>
        <input
          type="checkbox"
          checked={agreed}
          onChange={(e) => setAgreed(e.target.checked)}
        />
        I agree to terms
      </label>
      <p>Status: {agreed ? 'Agreed' : 'Not agreed'}</p>
    </div>
  );
}

Key Points:

  • Uses checked prop instead of value
  • e.target.checked gives the checkbox state
  • Conditional rendering shows different text based on state

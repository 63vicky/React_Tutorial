React Hooks - Student Practice Questions

Q1: useState - Counter App

Create a simple counter with two buttons: "+" and "-". Show current count in an <h1> tag.

Expected Output: The number should increase or decrease when buttons are clicked.

Q2: useEffect - Data Fetch

Fetch data from https://jsonplaceholder.typicode.com/users and display the list of names when the component mounts.

Expected Output: List of user names shown in a <ul>.

Q3: useRef - Input Focus

Create an input field and a button. On button click, the input should be focused.

Expected Output: Cursor automatically appears in the input box when the button is clicked.

Q4: useMemo - Expensive Calculation

Create an app that calculates factorial of a number. Use useMemo to avoid recalculating when not needed.

Expected Output: Factorial result updates only when the number input changes.

Q5: useCallback - Callback Optimization

Create a parent-child component where a memoized function is passed to the child using useCallback.

Expected Output: Child component doesn't re-render unnecessarily.

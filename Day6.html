⏱ 0:00 – 5:00 मिनट

Introduction & Conceptual Overview

React Router एक library है जो React में page reload के बिना navigation की सुविधा देता है। ये SPA के लिए जरूरी है।

Key Points:

  • Client-side routing
  • No page reloads
  • Maintains application state
  • Better user experience

⏱ 10:00 – 15:00 मिनट

BrowserRouter, Routes, Route

Import करना:

import { BrowserRouter, Routes, Route } from 'react-router-dom';

Components Explanation:

  • BrowserRouter: Main wrapper component
  • Routes: Contains all route definitions
  • Route: Individual route mapping

⏱ 15:00 – 25:00 मिनट

Basic Routing Code Example

function App() {
  return (
    <BrowserRouter>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
      </Routes>
    </BrowserRouter>
  );
}

⏱ 30:00 – 35:00 मिनट

Link Component

import { Link } from 'react-router-dom';

function Navbar() {
  return (
    <nav>
      <Link to="/">Home</Link> | 
      <Link to="/about">About</Link>
    </nav>
  );
}

Why Link not <a>?

Regular anchor tags cause page reload. Link components don't.

Nested Routing Concept

<Route path="/dashboard" element={<Dashboard />}>
  <Route path="profile" element={<Profile />} />
  <Route path="settings" element={<Settings />} />
</Route>

Dashboard component में <Outlet /> डालना जरूरी है: 

function Dashboard() {
  return (
    <div>
      <h2>Dashboard</h2>
      <Outlet />
    </div>
  );
}
